# Transferencia de datos
Es posible transferir datos de el sistema a un contenedor y viceversa. Es particularmente útil cuando queremos extraer un log, inyectar un archivo de configuración rápidamente o salvar datos de una base de datos que no tiene volúmenes configurados. Esto es posible con el comando `docker cp`.

## `docker cp`
Es la herramienta que te permite copiar archivos o carpetas entre el sistema de archivos de el sistema host y el sistema de archivos de un contenedor.

### Sintaxis
Del host al contenedor:

```bash
$ docker cp /ruta/en/mi/pc/archivo.txt nombre_contenedor:/ruta/en/contenedor/
```

Del contenedor al host:

```bash
$ docker cp nombre_contenedor:/ruta/en/contenedor/archivo.txt /ruta/en/mi/pc/
```
