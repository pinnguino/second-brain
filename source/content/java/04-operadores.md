### Operador `instanceof`
El operador `instanceof` nos permite comparar un objeto con una clase y determinar el tipo de objeto que es, retornando `true` o `false`.

```java
String s = "Java!";
Integer n = 2025;

boolean b1 = s instanceof String; // true
boolean b2 = n instanceof Integer; // true
boolean b3 = s instanceof Integer; // false
```
